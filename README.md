# Camppy

🌳 Camping makes you happy 🌳

### 👉🏻介紹
Camppy 是一款提供露營營地查詢的 LINE Bot。

* 透過下方 QRcode 或 ID 與 Camppy 成為好友。
* 輸入想查詢的縣市名稱，即可查詢位於該縣市的營區，目前有提供查詢的縣市如下：
    * 台北、新北、桃園、新竹、苗栗、台中、南投、雲林、嘉義、台南、高雄、屏東、宜蘭、花蓮、台東
